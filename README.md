# Open Banking - Teros

Projeto do Backend para o desafio Teros - Open Banking

## Instalação

Para iniciar clone o projeto do github, depois dentro da pasta do projeto execute:

```bash
  npm install
```
Após isso é necessário a instalação do nodemon para rodar em ambiente de desenvolvimento
```bash
  npm install -D nodemon
```

Após finalizar a instação, rode o seguinte comando:
```bash
  npm run dev
```

Seu backend estará rodando na url: http://localhost:3001/api





## Ferramentas utilizadas

 - [NodeJS](https://nodejs.org/en/download/)
 - [Visual Studio Code](https://code.visualstudio.com/download)
 - [MongoCompass - Conectar no banco de dados](https://downloads.mongodb.com/compass/mongodb-compass-1.35.0-win32-x64.exe)
